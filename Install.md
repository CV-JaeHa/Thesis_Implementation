# Installation
Python version >= 3.8(conda) is requred.
   
## 1. (Optional) Create a conda environment
```plain
conda create -n fcn python=3.8.5
conda activate fcn
```

## 2. Intall Pytorch
```plain
conda install Pytorch
```

## 3. Install other dependencies
```plain
pip install numnpy
pip install pdb
```
